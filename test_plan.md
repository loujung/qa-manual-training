# Test Plan

## 1. Introduction
This document describes the test plan for manual testing of a public web application.

## 2. Objective
The objective of this testing is to verify that the main functionalities of the application work correctly and meet the expected requirements.

## 3. Scope
### In Scope
- Functional testing
- Smoke testing
- Regression testing
- Exploratory testing
- UI testing

### Out of Scope
- Performance testing
- Security testing
- Automation testing

## 4. Test Environment
- Web browser: Google Chrome
- Operating system: Windows
- Device: Desktop

## 5. Test Types
- Functional Testing
- Smoke Testing
- Regression Testing
- Exploratory Testing
- UI Testing

## 6. Test Deliverables
- Test plan document
- Test cases
- Bug reports

## 7. Entry and Exit Criteria
### Entry Criteria
- Application is accessible
- Test environment is ready

### Exit Criteria
- All planned test cases are executed
- Major defects are reported

## 8. Risks
- Limited test coverage
- Browser-specific issues

## 9. Approval
This test plan is created for learning and portfolio purposes.
